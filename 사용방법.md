# hangulDisp 사용방법

이 문서는 통합 어댑터 기반의 간단한 예제를 제공합니다.

## 공통 준비

- hangulDisp 라이브러리
- 필요한 디스플레이 라이브러리 (u8glib/u8g2/gxepd2/tftlcd)
- 기본 H01 폰트는 `Font_name.h` 사용
- 폰트 변경 시 `hangul.setFont(폰트명)` 사용

## u8glib 예제 (OLED)

- 사용 가능한 색상: `0`(BLACK), `1`(WHITE), 또는 `HG_COLOR_BLACK`/`HG_COLOR_WHITE`

```cpp
#include <U8glib.h>
#include "hangulDisp.h"
#include "Font_name.h"

U8GLIB_SSD1306_128X64 u8g(U8G_I2C_OPT_NONE);

void drawPixel(int16_t x, int16_t y, uint16_t color) {
    u8g.setColorIndex(color ? 1 : 0);
    u8g.drawPixel(x, y);
}

hangulDisp hangul(drawPixel);

void setup() {
    u8g.setFont(u8g_font_6x10);
    hangul.setFont(Font_name);
}

void loop() {
    u8g.firstPage();
    do {
        hangul.print(0, 20, "안녕하세요 한글출력입니다", HG_COLOR_WHITE);
    } while (u8g.nextPage());
}
```

## u8g2 예제 (OLED)

- 사용 가능한 색상: `0`(BLACK), `1`(WHITE), 또는 `HG_COLOR_BLACK`/`HG_COLOR_WHITE`

```cpp
#include <U8g2lib.h>
#include "hangulDisp.h"
#include "Font_name.h"

U8G2_SSD1306_128X64_NONAME_F_HW_I2C u8g2(U8G2_R0, U8X8_PIN_NONE);

void drawPixel(int16_t x, int16_t y, uint16_t color) {
    u8g2.setDrawColor(color ? 1 : 0);
    u8g2.drawPixel(x, y);
}

hangulDisp hangul(drawPixel);

void setup() {
    u8g2.begin();
    u8g2.clearBuffer();
    hangul.setFont(Font_name);
    hangul.print(0, 20, "안녕하세요 한글출력입니다", HG_COLOR_WHITE);
    u8g2.sendBuffer();
}

void loop() {}
```

## gxepd2 예제 (E-Paper)

- 사용 가능한 색상: `GxEPD_BLACK`, `GxEPD_WHITE` (3색 모듈은 `GxEPD_RED`)

```cpp
#include <GxEPD2_BW.h>
#include "hangulDisp.h"
#include "Font_name.h"

GxEPD2_BW<GxEPD2_154, GxEPD2_154::HEIGHT> display(
    GxEPD2_154(SS, 8, 9, 7)
);

void drawPixel(int16_t x, int16_t y, uint16_t color) {
    display.drawPixel(x, y, color);
}

hangulDisp hangul(drawPixel);

void setup() {
    display.init();
    display.setFullWindow();
    hangul.setFont(Font_name);
    display.firstPage();
    do {
        display.fillScreen(GxEPD_WHITE);
        hangul.print(10, 30, "안녕하세요 한글출력입니다", GxEPD_BLACK);
    } while (display.nextPage());
}

void loop() {}
```

## tftlcd 예제 (TFT_eSPI)

- 사용 가능한 색상: `TFT_BLACK`, `TFT_WHITE`, `TFT_RED`, `TFT_GREEN`, `TFT_BLUE` 등

```cpp
#include <TFT_eSPI.h>
#include "hangulDisp.h"
#include "Font_name.h"

TFT_eSPI tft = TFT_eSPI();

void drawPixel(int16_t x, int16_t y, uint16_t color) {
    tft.drawPixel(x, y, color);
}

hangulDisp hangul(drawPixel);

void setup() {
    tft.init();
    tft.fillScreen(TFT_BLACK);
    hangul.setFont(Font_name);
    hangul.print(10, 30, "안녕하세요 한글출력입니다", TFT_WHITE);
}

void loop() {}
```
